## Contents

- `/data/me-sign-examples/`
  + drawn sample images to test glyph recognition, organized by gardiner number
- `/data/utf-pngs/`  
  + images of glyphs generated directly from utf hex codes
- `/data/gardiner_hieroglyphs_with_unicode_hex.json`
  + array of JSON "glyph" objects containing each glyph's utf hex code, utf character, and gardiner number along with descriptive information and a boolean `is_priority` value to designate priority status
- `/lib`  
  + scripts and resources for cleaning & producing the data
- `/docs`
  + additional documentation
